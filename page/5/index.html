word快速样式中的标题2标题3不见了怎么办一般情况下word快速样式中默认的有标题1，标题2，标题3和标题4，如果某一天你发现快速样式中的标题2和标题3，或是其他任意格式不见了，应该如何找回呢？请看下文。 方法/步骤 打开word，在开始菜单下的工具栏中找到样式，然后点击图示的下拉箭头。 展开快速样式以后，在窗口中看到并没有标题2和标题3，但是在word编辑过程中需要使用这两种快速样式，应该怎么办呢？继续看下文。 点击样式栏右下方图示位置的箭头符号。 点击之后会出现下图的样式窗口，在样式窗口底部点击管理样式图标，如图所示。 这时，弹出管理样式的设置页面，在该页面上方的菜单中点击【推荐】选项卡，然后在图示位置找到并选中标题2，接着点击下方的【显示】按钮。 这样标题2的状态就变成了显示了，按同样的方法对标题3进行显示设置操作，设置好之后点击窗口下方的【确定】。 点击确定后样式管理窗口消失，我们发现在工具栏的样式中标题2和标题3显示出来了，设置成功。

git出现RPC failed; curl 56 OpenSSL SSL_read: SSL_ERROR_SYSCALL, errfno 10054问题：git下载代码时，报“git出现RPC failed; curl 56 OpenSSL SSL_read: SSL_ERROR_SYSCALL, errfno 10054”的问题，如下所示的截图 解决步骤： 首先输入如下命令： 　　git config http.sslVerify “false” 如果输入上面命令提示报“ fatal: not in a git directory ”，则输入如下命令：git init 重新下载代码： git clone https://gitee.com…… 如果还是git代码还是下载失败，则需要修改git缓存的大小为100MB，输入如下命令： git config –global http.postBuffer 100M 问题解决了。
